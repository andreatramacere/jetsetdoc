Basic setup¶

A jet instance can be built using the the :class:`.Jet` class, istanciating the object in the following way:
[1]:

from jetset.jet_model import Jet
my_jet=Jet(name='test',electron_distribution='lppl',)
This instruction will create: * a ``Jet`` object with ``name`` **test**, * using as electron distribution the **lppl** model, that is a log-parabola with a low-energy power-law branch. (a working directory ``jet_wd`` will be created, this directory can be deleted when all the precesses of your script are done) For a list of possible distribution you can run the command
[2]:

Jet.available_electron_distributions()

lp: log-parabola
pl: powerlaw
lppl: log-parabola with low-energy powerlaw branch
lpep: log-parabola defined by peak energy
plc: powerlaw with cut-off
bkn: broken powerlaw
spitkov: spitkov
lppl_pile_up: log-parabola with low-energy powerlaw branch and pile-up
bkn_pile_up: broken powerlaw and pileup

custom electron distributions can be created by the user as described in this section of the tutorial :ref:`custom_emitters_guide`

Setting the parameters¶

assume you want to change some of the parameters in your model, you can use two methods: 1) using the :class:`.Jet.set_par()` method
[6]:

my_jet.set_par('B',val=0.2)
my_jet.set_par('gamma0_log_parab',val=5E3)
my_jet.set_par('gmin',val=1E2)
my_jet.set_par('gmax',val=1E8)
my_jet.set_par('R',val=1E15)
my_jet.set_par('N',val=1E3)
2) accessing directly the parameter
[7]:

my_jet.parameters.B.val=0.2
my_jet.parameters.r.val=0.4

Switching on/off the particle distribution normalization¶

As default the electron distributions are normalized, i.e. are multiplied by a constant ``N_0``, in such a way that : :math:`\int_{\gamma_{min}}^{\gamma_{max}} n(\gamma) d\gamma =1`, it means the the value `N`, refers to the actual density of emitters. If you want to chance this behavior, you can start looking at the sate of ``Norm_distr`` flag with the following command
[26]:

my_jet.Norm_distr

[26]:

1

and then you can switch off the normalization withe command

[27]:

my_jet.switch_Norm_distr_OFF()

OR

[28]:

my_jet.Norm_distr=0

[29]:

my_jet.switch_Norm_distr_ON()

OR

[30]:

my_jet.Norm_distr=1

Setting the particle density from observed Fluxes or Luminosities¶

It is possible to set the density of emitting particle starting from some observed luminosity or flux (see the method :meth:`.Jet.set_N_from_nuFnu`, and :meth:`.Jet.set_N_from_nuLnu`)
[31]:

my_jet=Jet(name='test',electron_distribution='lppl')

this is the initial value of N

[32]:

my_jet.parameters.N.val

[32]:

100.0
we now want to set the value of ``N`` in order that the observed synchrotron flux at a given frequency matches a desired value. For example, assume that we wish to set ``N`` in order that the synchrotron flux at :math:`10^{15}` Hz is exactly matching the desired value of :math:`10^{-14}` ergs cm-2 s-1. We can accomplish this by using the method :meth:`.Jet.set_N_from_nuFnu()` as follows:
[ ]:



[33]:


my_jet.set_N_from_nuFnu(nuFnu_obs=1E-14,nu_obs=1E15)

This is the updated value of N, obtained in order to match the given flux at the given frequency

[34]:

my_jet.get_par_by_name('N').val

271.77338679726074

my_jet.parameters.N.val

as you can see, the synchrotron flux at :math:`10^{15}` Hz is exactly matching the desired value of :math:`10^{-14}` ergs cm-2 s-1. Alternatively, the value of N can be obtained using the rest-frame luminosity and frequency, using the method :meth:`.Jet.set_N_from_nuLnu`
[38]:

my_jet.set_N_from_nuLnu(nuLnu_src=1E43,nu_src=1E15)
where ``L_0`` is the source rest-frame isotropic luminosity in erg/s at the rest-frame frequency ``nu_0`` in Hz.
